REPORTER: Hybrid Attention is a new architectural approach designed to significantly speed up inference in language models while maintaining low perplexity. In traditional full attention mechanisms, computational complexity scales quadratically with sequence length, making long contexts very expensive. This new Hybrid Attention model, which involves a linear first layer, a middle quadratic layer, and a last linear layer, dramatically reduces this to a near-linear complexity. For a Rust-focused language model, inference went from about 5.6 tokens per second with full attention to 286.6 tokens per second with Hybrid Attention, a 51x speedup. This is achieved through a combination of local windowed causal attention, a GRU-like recurrent state path, and an efficient KV cache paging strategy, allowing large speedups on consumer GPUs without visible quality loss. This research suggests a promising path for making advanced language models more accessible and efficient.
